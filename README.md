# THE LAST SAVE — تجربة ذاكرة

> لعبة تفاعلية عن ذكريات الألعاب وهويتك الرقمية

**العب الآن:** https://61465.github.io/game-zone-game

---

## ما هي اللعبة؟

THE LAST SAVE تجربة سردية فريدة: النظام ينهار وكل ذكرياتك من الألعاب تُحذف. مهمتك صد الهجمات بالكتابة السريعة واختيار ما تحفظه — لكن الثمن هويتك نفسها.

## الأسلوب

- **اكتب الكود** لصد كل هجوم (حروف A S D F J K L M)
- **احفظ** ذاكرة بعد الصد بضغط S
- **أتركها تذهب** بضغط D
- كل ذاكرة محفوظة تكلفك نقاط هوية

## الألعاب في اللعبة

- Minecraft · GTA V · The Witcher 3 · Red Dead Redemption 2 · Elden Ring

## التقنية

- HTML + CSS + JavaScript (ملف واحد، بدون مكتبات)
- Web Audio API للمؤثرات الصوتية
- Canvas API لحقل النجوم
- localStorage لحفظ أفضل نتيجة

## تشغيل محلي

افتح `index.html` مباشرة في المتصفح — لا يحتاج خادم.
